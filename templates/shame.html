Wall of Shame

The following bioRxiv preprints appear to have been posted after acceptance, which is not only deceptive and bad practice, but against bioRxiv policies. Unfortunately these are not the only preprints deserving of shame. Michael Eisen has said that preprints without methods should just be called "ads". John Didion has suggested that these be called "TEDprints". I would also include preprints which don't have data or code in these categories, but I don't have a means for systematically finding "TEDprints", so I can't include them in this list.
